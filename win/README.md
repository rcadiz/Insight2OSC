Para WIN:
Para realizar el auto ejecultable copiar el archivo o agregar al path C:\Python27\Scripts\pyinstaller, y ejecutar por consola desde la carpeta que contiene Window.py: pyinstaller Window.py el archivo ejecutable estará en la carpeta dist.

Descargar PyQt4 desde el siguiente link: 
https://www.riverbankcomputing.com/software/pyqt/download
Si es desde Window elegir el archivo:
PyQt4-4.11.4-gpl-Py2.7-Qt4.8.7-x32.exe

Abrir la consola en C:/Python27/Scripts y ejecutar:
pip install PyInstaller
pip install pyOSC

Para MAC:
Descargar..


Para crear el autoejecutable abrir la consola donde esté el archivo main.py, y ejecutar:
pyinstaller main.py

El archivo se guardará en la carpeta dist.
